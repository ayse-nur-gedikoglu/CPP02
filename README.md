Ad-hoc polymorphism, operator overloading and the Orthodox Canonical class form

ex00: Orthodox Canonical Class Form Nedir?

Doğru ve güvenli nesne kopyalama ve imha sağlayan üç özel üye işlevinden oluşan standart settir:

- Copy constructor
MyClass(const MyClass& other);


- Copy assignment operator
MyClass& operator=(const MyClass& other);

- Destructor
~MyClass();
